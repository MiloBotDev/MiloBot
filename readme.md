### !prefix
Change the prefix of the guild you're in.
#### Usage
`;prefix {prefix}` 
##### Permissions
`ADMINISTRATOR`

### !usage
See the amount of times each command has been used.
#### Usage
`!usage {*command}`
#### Cooldown
60 seconds.
